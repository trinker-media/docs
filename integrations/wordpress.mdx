## Overview

The WordPress integration lets you connect your WordPress site to Genseo using the official Genseo plugin and a Project API Key.

Once connected, you can publish and update articles directly from Genseo.

<video controls className="w-full aspect-video rounded-xl" src="https://videos.genseo.co/integrations/genseo-integration-wordpress.mp4" />

## Before You Start

- You need admin access to your WordPress dashboard
- You need access to your Genseo account at [app.genseo.co](https://app.genseo.co)
- You need permission to install and activate WordPress plugins

## Connect WordPress in Genseo

### Step 1: Open WordPress

1. Open your WordPress admin dashboard

### Step 2: Install the Genseo Plugin

1. In the WordPress sidebar, go to **Plugins** → **Add Plugin**
2. Search for **Genseo**
3. On the **Genseo** plugin card, click **Install Now**
4. Click **Activate**

<Tip>
You can also open the plugin directly in the WordPress Plugin Directory: [wordpress.org/plugins/genseo](https://wordpress.org/plugins/genseo)
</Tip>

### Step 3: Open the Genseo Plugin in WordPress

1. In the WordPress sidebar, click **Genseo**

### Step 4: Create an API Key in Genseo App

1. Open [app.genseo.co](https://app.genseo.co)
2. Go to **Integrations**
3. Select **WordPress**
4. Create a new API Key

### Step 5: Connect WordPress to Genseo

1. Copy the API Key from Genseo
2. Go back to WordPress → **Genseo**
3. Paste the key into **Project API Key**
4. Click **Connect to Genseo**

### Step 6: Verify the Connection

1. Return to [app.genseo.co](https://app.genseo.co)
2. In **Integrations** → **WordPress**, click **Check connection**
3. Confirm the success message

Your WordPress site is now connected and ready for publishing.

## Common Issues and Solutions

### Genseo Plugin Not Found in WordPress

**Problem**: You cannot find the Genseo plugin in **Plugins** → **Add Plugin**.

**Solution**:
- Confirm you searched for `Genseo` in the plugin search field
- Check that your WordPress site can access the WordPress Plugin Directory
- Open the direct plugin page: [wordpress.org/plugins/genseo](https://wordpress.org/plugins/genseo)
- Make sure your WordPress role has permission to install plugins

### Invalid API Key

**Problem**: Connection fails after pasting the key.

**Solution**:
- Create a new API Key in Genseo and try again
- Copy and paste the full key without extra spaces
- Ensure you are using a WordPress key from **Integrations** → **WordPress**

### Check Connection Fails

**Problem**: **Check connection** does not pass in Genseo.

**Solution**:
- Confirm the plugin is installed and activated in WordPress
- Re-open WordPress → **Genseo** and click **Connect to Genseo** again
- Verify the same API Key is active in Genseo
- Refresh both dashboards and re-run **Check connection**

### Missing Permissions in WordPress

**Problem**: You do not see **Plugins** or cannot activate/install plugins.

**Solution**:
- Ask a WordPress administrator to grant plugin management permissions
- Complete plugin installation with an admin account
- Then continue connection in WordPress → **Genseo**

## Best Practices

- **Use one key per project**: Keep project environments clearly separated
- **Rotate keys when needed**: Create a new key immediately if one is exposed
- **Test before bulk publishing**: Publish one test article after connecting
- **Verify result in WordPress**: Check formatting, media, and slug after publish
- **Keep plugin updated**: Install plugin updates for compatibility and stability
